# free-hls.js

一个用 Node.js 写的 Free-HLS 上传客户端

## Requirements
- NodeJS
- Yarn
- FFmpeg

## Installation

```bash
git clone https://github.com/sxzz/free-hls.js.git
cd free-hls.js
yarn install
yarn upload <file> [title]
```
